I membri di Rosé Pine incoraggiano la creatività e le interpretazioni diverse della nostra tavolozza limitata per i temi della community. Se preferisci un approccio più strutturato con una guida lungo il percorso, accogliamo i temi ufficiali nell'organizzazione GitHub di Rosé Pine, dove il team aiuterà quando possibile.

Inizia con il nostro [template del repository](https://github.com/rose-pine/rose-pine-template) come base per il tuo nuovo tema.

```sh
gh repo create my-app --public --clone --template rose-pine/rose-pine-template
```

Usa il nostro strumento di build, [bloom](https://github.com/rose-pine/rose-pine-bloom), per accedere facilmente alla nostra palette e includere tutte e tre le varianti. Crea un template da un file di tema esistente tramite `bloom init <filename>`.

Una volta completato, puoi [inviare il tuo tema](/submit-a-theme) come tema ufficiale o della community. Per i temi ufficiali, ti inviteremo nell'organizzazione GitHub per il trasferimento. Il tema verrà aggiunto automaticamente al sito. Per i temi della community, il modulo di invio è il modo in cui aggiungiamo il tuo tema al sito. Non sono necessarie ulteriori azioni.

Grazie per aver condiviso con la community, non vediamo l'ora di vedere cosa crei!
