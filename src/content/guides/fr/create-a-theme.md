Les membres de Rosé Pine encouragent la créativité et les interprétations variées de notre palette limitée pour les thèmes de la communauté. Si vous préférez une approche plus structurée avec un accompagnement tout au long du processus, nous accueillons volontiers les thèmes officiels au sein de l'organisation GitHub Rosé Pine, où l'équipe vous aidera dans la mesure du possible.

Commencez par utiliser notre [modèle de dépôt](https://github.com/rose-pine/rose-pine-template) comme base pour votre nouveau thème.

```sh
gh repo create my-app --public --clone --template rose-pine/rose-pine-template
```

Utilisez notre outil de build, [bloom](https://github.com/rose-pine/rose-pine-bloom), pour accéder facilement à notre palette et inclure les trois variantes. Créez un modèle à partir d'un fichier de thème existant via `bloom init <filename>`.

Une fois le build terminé, vous pouvez [soumettre votre thème](/create/submit-a-theme) en tant que thème officiel ou de la communauté. Pour les thèmes officiels, nous vous inviterons à rejoindre l'organisation GitHub afin que vous puissiez y transférer votre dépôt. Celui-ci sera alors automatiquement ajouté au site web. Pour les thèmes de la communauté, la soumission de votre formulaire suffit pour que nous ajoutions votre thème au site. Aucune action supplémentaire n'est requise.

Merci de partager votre travail avec la communauté, nous avons hâte de voir ce que vous allez créer!
